  • [mac]FUSE for OS X をインストール
    • 目的は２つある。
      • Raspberry Pi の SD カードをマウントして、中身を見たい。
        • RPi は linux なので、 ext2 とか ext3 とかを OS X でマウントしたい、 ということになる。
      • FreeBSD で使ってた disk をマウントしたい。
        • FreeBSD は、最近 zfs というものに移行したが、それ以前は ufs (ufs2) いわゆる unix file system.
    • ext2 を読めるようにする。
      • ACCESS EXT3/EXT2 FILE SYSTEM ON MAC OSX LION (10.7) / Posted by Nirupam Biswas on Saturday, December 3, 2011
        • 情報はちょっと古いが、Mac 用の FUSE を入れて、 ext2 プラグインを入れればいい、ということらしい。
      • How to manage Ext2/Ext3 disks in OS X / by Topher Kessler June 22, 2012 9:54 AM PDT
        • こっちはもうちょっと新しい。内容はほぼ一緒。（MacFUSE というのが更新停止してて、OSXFUSE を使ったほうがいいらしい、と）
      • OSXFUSE: http://osxfuse.github.io/
        • download: osxfuse-2.7.4.dmg
      • fuse-ext2: http://sourceforge.net/projects/fuse-ext2/
        • fuse-ext2-0.0.7.dmg ってのをダウンロード。
      • 注意点： fuse-ext2 は OSXFUSE じゃなくて古い MacFUSE 用なので、 OSXFUSE インストール時に後方互換を入れておく必要がある。
      • screenshots
    • FreeBSD について
      • stackexchange: Reading FreeBSD UFS formatted external drive in Mac OS X
      • Mount a ufs2 Volume in MacOS/X 10.7 (Lion) / Created by Thomas Gutzmann, last modified on 2013-02-22
