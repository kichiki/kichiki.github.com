      • glXMakeCurrent() を OSMesaMakeCurrent() に変更
      • Initialize() と Destroy() を適切に処理
      • Initialize() での注意事項
        • X も取得だけはしておく
        • OSMesa, X ともに、それぞれ属性の設定を行う
        • cabrales (Linux) では Pixmap の生成は glXCreateGLXPixmapMESA() を使わないとダメだった
    • 使ってみたい方はどうぞ
      • 元にした vtkOffScreen の snapshot と、そいつに当てる patch
      • 具体的な遊び方 (上の他に、当然 vtk 自身も必要):
        • tar xvfz vtkOffScreen.tar.gz
        • cd vtkOffScreen
        • patch <../vtkOffScreen-OSMesa.patch
        • Makefile を修正して、たとえば make vtkText_ex としてみる

    $fh = 'fh'.$i;

    if (eof (fh.$i)) {

    }
    }
