apsrev-title.bst (BibTeX style file for Phys. Fluids)

apsrev-title.bst is a BibTeX style file for REVTeX 4 with the title on each article:
(Example) A.B.Cdefg and H.Ijkl, "A paper for physics of fluids," Phys. Fluids 16, 1234 (2004).
This is for the recent Physics of Fluids.

apsrev-title.diff is the patch to apsrev.bst (*** REVTeX-compatible apsrev.bst 1702 ***).
