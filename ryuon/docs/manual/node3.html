stokes: Stokesian Dynamics Simulator

RYUON-stokes is a package of applications using libstokes;
  • stokes3 : Stokesian dynamics simulator
  • xi3 : Tuning program for the Ewald summation
  • Visualization and other utilities
  • Wrappers for the scripting languages, Python, Perl, Ruby, Guile, and Octave.
